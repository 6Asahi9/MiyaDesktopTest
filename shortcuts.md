# General

- ESC:
  When inside Music, Neon Picker, or Add Application, ESC closes the current page and returns to the main window
- BackSpace:
  minimizes the window

1 — Open Add Application Page
2 — Open Music Page
3 — Open Choose Neon Colour
4 — Open Custom

# Add Application

    Double-tap a row to open the edit window
    Tap + Delete on a row to remove the application from the container

1 - Open Add Application window
2 - Auto Add Applications

# Music

    Music are on repeat by default
    Use arrows keys < > to move between different musics
    Use arrows keys up and down to contol the volume
    Press enter to play/pause music

# Playback Modes in Music

1 - puts music on repeat
2 - puts music on juggle
3 - puts music on straight
4 - opens the music list
